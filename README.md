# AgroMart - Your Farming Partner

Agricultural shop PWA for farmers. Buy seeds, fertilizers, pesticides and tools. Available in English and Tamil.

## Getting Started

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm i

# Start the development server
npm run dev
```

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- PWA with offline support

## Deployment

Deploy via your preferred hosting platform. Run `npm run build` to generate the production build.
